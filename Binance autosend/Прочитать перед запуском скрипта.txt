1. Убедитесь, что у вас установлен python.

Чтобы узнать установленную версию Python, выполните следующие действия:
- Откройте командную строку Windows или терминал на Ubuntu.
- Введите команду "python --version" (без ковычек) 

После этого вы должны увидеть вывод информации о версии Python, которая установлена на вашем компьютере. Например, вывод может быть следующим:
"Python 3.9.4"
Если Python не был найден, проверьте, был ли он установлен правильно, и убедитесь, что вы вводите правильную команду.
Если он не установлен, следуям инструкциям ниже.

Установка Python на Windows:

Скачайте установщик Python для Windows с официального сайта по ссылке https://www.python.org/downloads/windows/
Запустите установщик Python и следуйте инструкциям мастера установки.
При установке убедитесь, что опция "Add Python to PATH" выбрана.
После завершения установки откройте командную строку и введите команду python. Если Python установлен правильно, вы должны увидеть версию Python, которую вы установили.

Установка Python на Ubuntu:

Откройте терминал в Ubuntu.
Убедитесь, что у вас установлен менеджер пакетов apt. Если нет, выполните команду sudo apt-get update для обновления пакетов.
Введите команду sudo apt-get install python3 для установки Python 3. 


Обратите внимание, что для успешной установки Python на обеих операционных системах необходимо иметь права администратора.


2. Введите команду "pip install python-binance" (без ковычек) для установки библиотеки binance для python.

3. Распакуйте архив с файлами

4. Заполните текстовый файл "addresses.txt" своими адресами, на которые будете отправлять крипту, по одному адресу на строку без ковычек и лишних знаков.

5. В файле "Binance.py":
- Вместо ТВОЙ_API_KEY и ТВОЙ_API_SECRET введите API Key и Secret Key со страницы https://www.binance.com/ru/my/settings/api-management сохраняя ковычки
- В строках с переменными ASSET_TO_WITHDRAW и NETWORK замените BTC на нужную крипту и сеть для вывода, сохраняя ковычки.
- Установите нужный вам диапазон и число знаков после запятой, заменив 2, 2.5 на свой диапазон, а 8 на нужное количество знаков после запятой в строке amount_to_withdraw = round(random.uniform(2, 2.5), 8)
- В строке time.sleep(random.randint(5, 9)) можете установить время сна в своем интервале (по умолчанию от 5 до 9 секунд между снятиями), можете указать свой для ускорения или замедления процесса 

6. Перед запуском скрипта перейдите в командной строке или терминале в директорию со скриптом:
cd ВАШ_ПУТЬ_К_СКРИПТУ

7. Запускаем срипт командой:
python Binance.py
Или для третьей версии python:
python3 Binance.py

Если много адресов и боитесь, что скрипт прервется, то можете запускать например в screen режиме. 
Наблюдаем за выводом!
